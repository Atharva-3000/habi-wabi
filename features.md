# Habit Tracker App - Features

## 1. Onboarding
- **Step-by-step Guide:** Carousel-style dots to guide the user seamlessly upon first installation.
- **Open and Free:** No login required. Fully open-source foundation, taking the user directly to the app without friction.

## 2. Core Habit Tracking
- **Habit Creation:**
  - **Inputs:** Title, Time of day.
  - **Reminders:** Toggle notifications (Yes/No).
  - **Frequency:** Choose specific dates, weekdays, or everyday.
  - **Color Selection:** User selects an accent color map for each specific habit. 
- **GitHub-Style Contribution Maps:**
  - Habits are visualized via small square grids where each square equals one day.
  - **Status Colors:** If done, the square fills with the habit's chosen color (e.g., bold green). If incomplete, it remains light tinted or grey.

## 3. Diet & Health Add-ons
- **Weight Tracking:** Input and monitor body weight over time.
- **Water Intake Tracker:** Easily track daily glasses or quantity of water.

## 4. Homescreen Experience
- **Daily Quotes:** A random motivational quote appears on the homescreen for every new day.
- **Quote Design:** Housed in a big elevated card or background, using a Calligraphy-like font inside large quotation marks.

## 5. To-Do and Calendar System
- **To-Do Lists:** Dedicated page for general tasks.
- **Calendar Integration:** Select any date on a built-in calendar to view or create specific to-dos.
- **Task Frequency:** Set to-dos to be weekly, daily, or custom intervals.

## 6. Settings
- **Basic Settings Page:** Allows tweaking of preferences, managing theme/notifications, and viewing basic app info.
