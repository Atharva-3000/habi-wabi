# Changelog

All notable changes to the **Habi Wabi** project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-09

### Added
- **Initial App Launch**
- Beautiful Splas Screen with multi-stage drawing animations (Enso arc and typography).
- Home Screen dashboard with live Habit tracking arcs.
- "Quick Log" Weight entry direct from the Home Screen with interactive bottom sheet.
- Interactive Water Tracking screen with visual wave fill animations and multiple vessel sizes.
- Full To-Do List functionality including grouping by date, filtering, and swipe-to-delete.
- Create Habit Flow with custom icons, colors, frequency (Daily/Weekly/Custom), and daily completion targets.
- Local Push Notifications for Habit reminders (via AlarmManager).
- Comprehensive Settings Screen: Global notification toggles, quiet hours definition, daily water goal setting, and 30-day weight history chart.
- Complete 100% on-device data persistence using Room Database and Jetpack DataStore.
- Forced distraction-free Dark Mode aesthetic.
